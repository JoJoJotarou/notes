
## AspectJ Maven Plugin

这个 Maven 插件使用 AspectJ 编译器（“ajc”）将 AspectJ 方面代码编织到您的 .class 文件中。static 或 final 的方法对它也是有效的。

- <https://github.com/mojohaus/aspectj-maven-plugin>
