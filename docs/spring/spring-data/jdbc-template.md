
## JDBCTemplate

```xml
<!-- jdbc api -->
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-jdbc</artifactId>
</dependency>
```

all-in-one 的依赖项

```xml
<dependency>
  <groupId>org.springframework.data</groupId>
  <artifactId>spring-data-jdbc</artifactId>
  <version>2.4.2</version>
</dependency>
```
